A plugin to add a button to flush permalinks in WordPress admin bar.

== Installation ==
1. Download the plugin
2. Login to your WordPress website dashboard
3. Navigate to *Dashboard* > *Plugins*
4. Click on *Add New* and then *Upload Plugin*
5. Browse to the file, select it and click *Install Now*
6. Click *Activate Plugin* once prompted

== Usage ==
Just click on the *Flush Permalinks* button in the WordPress admin bar.
